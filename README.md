> npm package dependencies visualizer

## Installation
```sh
$ npm i
```

## Usage
### dev
```sh
$ npm start
```

### prod-ish
* build it
```sh
$ npm run build
```

* inspect the builded bundle content through interactive zoomable treemap here: `/dist/report.html`

* run it locally
```sh
$ npm run serve:prod
```
